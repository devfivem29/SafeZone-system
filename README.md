# SafeZone-system
Ce script permet d'ajouter des zones sécurisées entièrement configurables sur un serveur FiveM
Ce script permet d'ajouter des zones sécurisées entièrement configurables sur un serveur FiveM. Les joueurs peuvent :

Ajouter et personnaliser des zones via le fichier config.lua (nom, coordonnées, rayon, etc.).
Configurer les blips (couleur, transparence, nom) pour chaque zone.
Ajuster les notifications textuelles (couleur, taille, position à l'écran).
Garantir la sécurité en empêchant l'utilisation d'armes et les attaques dans ces zones.
Simple à configurer, ce script est parfait pour les serveurs RP souhaitant offrir des espaces sûrs à leurs joueurs
